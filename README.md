# Trading TradingView Scripts

Collection of TradingView Pine Script indicators for ICT (Inner Circle Trader) concepts.

## Scripts

- **ICT_Crypto_Displacement_Block_v6.pine** - Displacement block indicator for crypto markets

## Usage

These scripts are designed for use on TradingView platform. To use:

1. Open TradingView
2. Open Pine Editor
3. Copy script content
4. Save and add to chart

## Concepts

These indicators implement ICT trading concepts including:
- Displacement blocks
- Market structure analysis
- Higher timeframe context
